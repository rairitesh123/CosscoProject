Technical Aptitude

Introduction Venue Instructions Question Paper

Technical Aptitude is the preliminary round for the event “Coder’s Cup”. This test is basically to find all those candidates who are strong with their technical knowledge in programming. Test contains questions based on ‘C’ Language. It is open for all students of Computer Science & Engineering branch. The top scorers get entry to the main technical coding event i.e. Coder’s Cup.
